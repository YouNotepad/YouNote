Problem & Target User

As a Computer Science student, Youtube is a very precious source to learn programming languages. However, we faced difficulties while watching youtube and taking notes at the same time. We struggled with pressing the alt - tab hundreds of times to switch video, note, and sometimes IDE. Therefore, we design an app for Computer Science students, who learns programming languages through Youtube.

Solution

Our solution YouNote sticks the video and notepad in a single page. Especially, since this product is designed for CS students, this app also supports code block function. Time stamping function supports the user to watch several videos in a single note. Moreover, users can adjust the size of video and note by dragging. Plus, this app supports various note styling of course.
